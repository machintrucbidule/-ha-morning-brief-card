# Morning Brief Card

Lovelace card that renders the canonical JSON produced by the
[`ha-morning-brief`](https://github.com/machintrucbidule/ha-morning-brief)
integration.

- Header with history navigation (← → ⟳)
- Alerts banner (severity-coloured)
- Per-category field lists with comparisons + sparklines
- Weather block + AI verdict
- Compact mode + 7 visibility toggles
- Themable via standard HA CSS variables, optional `theme_override`
- FR + EN, language follows `hass.language`

See the README for installation and configuration.
